NEXT STEPS - SETUP TUNNEL
================================================================================

Step 1: Authenticate with Cloudflare
────────────────────────────────────

Run this command:

    C:\Users\bhavi\cloudflared.exe login

Or (if added to PATH from new terminal):

    cloudflared login

This will:
  1. Open your browser
  2. Ask you to sign in to Cloudflare
  3. Select your domain
  4. Generate credentials file at: C:\Users\bhavi\.cloudflared\cert.pem

✅ Do this next!

────────────────────────────────────────────────────────────────────────────

Step 2: Create Named Tunnel
────────────────────────────

    C:\Users\bhavi\cloudflared.exe tunnel create federated-backend

Output will show:
  - Tunnel ID: abc123def456...
  - Credentials file: C:\Users\bhavi\.cloudflared\TUNNEL_ID.json

Save the Tunnel ID! ⭐

────────────────────────────────────────────────────────────────────────────

Step 3: Create config.yml
──────────────────────────

Create file: C:\Users\bhavi\.cloudflared\config.yml

Content:

    tunnel: YOUR_TUNNEL_ID
    credentials-file: C:\Users\bhavi\.cloudflared\YOUR_TUNNEL_ID.json

    ingress:
      - hostname: api.YOUR_DOMAIN.com
        service: http://localhost:8002
      
      - service: http_status:404

Replace:
  - YOUR_TUNNEL_ID: Your actual tunnel ID
  - YOUR_DOMAIN: Your domain (e.g., example.com)

────────────────────────────────────────────────────────────────────────────

Step 4: Create DNS Record
──────────────────────────

1. Go to: https://dash.cloudflare.com
2. Select your domain
3. DNS → Add Record:
   Type: CNAME
   Name: api
   Content: YOUR_TUNNEL_ID.cfargotunnel.com
   Proxy: Proxied (orange cloud)
4. Save

────────────────────────────────────────────────────────────────────────────

Step 5: Run the Tunnel
──────────────────────

    C:\Users\bhavi\cloudflared.exe tunnel run federated-backend

You should see:
  ✅ "Tunnel registered with ID: abc123def456..."
  ✅ "Tunnel running successfully"

────────────────────────────────────────────────────────────────────────────

Step 6: Test It Works
──────────────────────

From any device/browser:

    https://api.YOUR_DOMAIN.com/health

Should return:

    {"status": "healthy", "mongodb_connected": true, ...}

✅ You're live!
